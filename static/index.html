About eyeBOLD
What is eyeBOLD?

eyeBOLD is a user-friendly web interface for querying and exploring DNA barcoding data. It integrates taxonomic, geographic, and sequence data to help researchers and educators access and analyze biodiversity information effortlessly.

Data Sources

eyeBOLD aggregates data from:

  • BOLD (Barcode of Life Data Systems): The global repository for DNA barcoding data.
  • GBIF (Global Biodiversity Information Facility): Open-access biodiversity data from around the world.
